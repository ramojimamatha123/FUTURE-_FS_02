
## Project Overview
This project is a simple CRM (Customer Relationship Management) application developed to manage client leads generated from website contact forms.

## Features
- Add new client leads
- Store lead information (Name, Email, Source, Status)
- Update lead status (New, Contacted, Converted)
- Add notes and follow-ups for each lead
- User-friendly dashboard
- Responsive design

## Technologies Used
- Frontend: HTML, CSS, JavaScript, React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## Purpose
The system helps businesses organize and track potential customers efficiently, improving lead management and follow-up
